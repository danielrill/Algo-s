# Dynamic Programming

This folder contains dynamic programming implementations in C.

Included algorithms:
- Fibonacci
- 0-1 Knapsack
- Longest Common Subsequence
- Longest Increasing Subsequence
- Matrix Chain Multiplication
- Edit Distance
- Rod Cutting
- Floyd-Warshall
- Coin Change
- Subset Sum DP
- Traveling Salesman Problem (bitmask DP)
